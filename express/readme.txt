HTTP
安装一个能显示HTTP 请求状态码和所有重定向的浏览器插
件。这样在解决重定向问题或者不正确的状态码时会更加容易，它们经常被
忽视。对于Chrome 来说，Ayima 的Redirect Path 特别好用。在大多数浏览
器中， 都能在开发者工具的网络部分看到状态码。
